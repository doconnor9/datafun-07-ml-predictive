Create a lambda to convert Fahrenheit temperatures to Celsius; store each pair as a tuple¶

In [ ]:
c = lambda f: 5 / 9 * (f - 32)

temps = [(f, c(f)) for f in range(0, 101, 10)]  # temps from 0-100 in 10 degree increments

Method comparison¶

Both methods resulted in similar values. The sklearn method uses simpler code to predict/estimate values. I liked the ability to use hue with Seaborn's scatterplot function. It seems like that might allow better visualization for certain datasets rather than using Seaborn's regplot.
